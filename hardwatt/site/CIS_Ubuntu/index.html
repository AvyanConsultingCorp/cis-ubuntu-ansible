CIS Security Bencmarks for Ubuntu

Recommendations

This recommendations provide prescriptive guidance for system and application administrators who plan to develop, deploy, assess, or secure solutions that incorporate Ubuntu server.

1 - Patching and Software Updates

Newer patches may contain security enhancements that would not be available through the latest full update. As a result, it is recommended that the latest software patches be used to take advantage of the latest functionality.

2 - Filesystem Configuration

Directories that are used for system-wide functions can be further protected by placing them on separate partitions. This provides protection for resource exhaustion and enables the use of mounting options that are applicable to the directory's intended use. It is recommended to store user's data on separate partitions and have stricter mount options.
